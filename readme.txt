# NYTIMES-BOOKS-APP
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

NYTIMES-BOOKS-APP is a React Based Web Application to get the the new york times best selling books.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
##Requirements
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
Node.js
Visual Studio Code
Web Browser

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
## Installation
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

1.Use the GitHub link to download full project file [zip](https://github.com/vidath96/nytimes-books-app.git).

2.Extract the zip file to a anywhere on your local disk.(D:\Sample-Folder\nytimes-books-app)

3.Run Web Application(Two Methods)

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
#### METHOD 1
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

>>Go inside project directory and press 'Sfift+Right Mouse Button'.
>>Click 'Open Powershell Window here' to open Powershell window.
>>Then type 'npm start' to start development server to run application.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
#### METHOD 2
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------
>>Go inside project directory and press 'Sfift+Right Mouse Button'.
>>Click 'Open Powershell Window here' to open Powershell window.
>>Then type 'code .' to open project with Visual Studio Code.
>>Go to terminal tab on Visual Studio Code.
>>Get New Terminal.
>>Type 'npm start' to run application.

Now Web Application is successfully running....!
